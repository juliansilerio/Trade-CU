# Trade @ CU
---
Simple trading system for students to exchange seats in classes.

### Requirements
* Node.js (I used 8.10.0)
* npm  (I used 6.9.0)
* Python3
  * flask
  * flask_sqlalchemy
  * flask_cors

### Installation
1. Install Python requirements
2. `npm install`

### Operation
1. python server/app.py
2. (navigate to server/client) npm start
3. `localhost:8080` to login
